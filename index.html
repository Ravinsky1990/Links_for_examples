Примеры работ

В примере используется сетка Bootstrap 3, @keyframes - анимация, плагины jquery для плавного скрола, динамической подгрузки элементов. Также добавлено адаптивное меню написаное на чистом jquery/css3 без использования плагинов.
В примере используеться Flexbox для создания сетки елементов с продуктами, также слайдер написан на чистом jquery/css3 без использования плагинов.
Добавлени счетчик времени, написанный на чистом JS
Используеться кастомизация селектов и подсказки при клике на поле для ввода.
